# GitScope 

A modern GitHub Profile Viewer built using HTML, CSS, and JavaScript.

## Features
- GitHub profile search
- Animated background
- Dark/Light theme
- Responsive UI
- Dynamic profile stats
- Social links support
- Smooth animations

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- GitHub REST API
